What is Halium?

Halium is a collaborative project in order to unify the HAL for projects, which bring GNU/Linux to mobiles with preinstalled Android.

Using it enables the usage of different RootFS of different GNU/Linux distributions, i.e. it has to be ported only once.

Halium contains:

  • Linux kernel (source code by OEM)
  • Android services for talking with the hardware (runs inside a LXC container)
  • Libhybris for drivers

Next to that, Halium tries to standardise middleware:

  • OFono, RILd (telephoby, SMS)
  • camera
  • PulseAudio, AudioFingerGlue
  • GPS
  • Media Codecs

Why is Halium needed?

Custom ROM developers are tasked to rewrite HAL code time and again. Halium creates a common base.

Triggered by shutting down of alternatives to Android (Firefox OS, Ubuntu Touch).

What is the current state of Halium?

You can see an Ubuntu 16.05 with systemd as init system. On top of it runs an Andoid 5 inside a LXC container and some LibHybris tests are passing.
